Exercício 2: Criando Listas Ordenadas

  1. Crie uma lista ordenada que represente as "Etapas de Desenvolvimento de um Projeto".

    2. A lista deve conter os seguintes itens na ordem correta:

    1. Definição de Requisitos
    2. Planejamento
    3. Design
    4. Implementação
    5. Testes
    6. Implantação
    7. Manutenção
  2. Crie uma lista ordenada chamada "Lista de Prioridades".

    3. A lista deve conter os seguintes itens na ordem correta:

    1. Finalizar o relatório de vendas
    2. Preparar a apresentação para a reunião
    3. Revisar o código do projeto
    4. Atualizar o banco de dados
    5. Enviar o feedback para a equipe

Exercício 3: Listas Aninhadas

  1. Crie uma lista não ordenada que represente "Itens para Viagem", contendo outras listas não ordenadas dentro dela.

    2. A lista principal deve ter os seguintes itens:

    • Documentos
      • Passaporte
      • Carteira de Identidade
      • Vistos
    • Roupas
      • Camisetas
      • Calças
      • Sapatos
    • Eletrônicos
      • Carregador de celular
      • Fones de ouvido
      • Adaptador de tomada
  2. Crie uma lista ordenada chamada "Preparação de uma Receita", contendo outras listas ordenadas dentro dela.

    3. A lista principal deve ter as seguintes etapas:

    1. Ingredientes
      1. Farinha de trigo
      2. Açúcar
      3. Ovos
    2. Preparo
      1. Misturar os ingredientes secos
      2. Adicionar os ovos e misturar bem
      3. Colocar a massa na forma
    3. Cozimento
      1. Pré-aquecer o forno a 180°C
      2. Assar por 30 minutos
      3. Deixar esfriar antes de servir
